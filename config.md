# Bulk Clear Formatting Configuration

## Settings

### show_in_browser_context_menu
- **Type**: Boolean (true/false)
- **Default**: true
- **Description**: Show "Clear Formatting..." option when right-clicking on notes/cards in the browser.

### show_in_editor_context_menu
- **Type**: Boolean (true/false)
- **Default**: true
- **Description**: Show clear formatting options when right-clicking in the note editor.

## How to Use

1. Set either option to `false` to hide that context menu entry

3. Restart Anki after changing these settings
